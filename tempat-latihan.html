Latihan 3 Elemen pada HTML

  • Elemen pada HTML

    • Tag : untuk bagian dari website.

    Atribut : Informasi tambahan yang ditulis kedalam tag pembuka.

    Value : Nilai dari atribut.


    Contoh
    < p > elemen pada html < /p> contoh tag (paragraf)

    < p align = "center" > elemen pada html < /p >

    Penjelasan
    • < p > adalah tag
    • align adalah atribut
    • "center" adalah value
    Contoh Penggunaan :

Contoh Penggunaan

Tanpa warna

Code : < p align = "center">Contoh Penggunaan< /p >

Contoh Penggunaan

Berwarna dan memakai atribut css

Code : < p style = "text-align:center;color: red;">Contoh Penggunaan< /p >

Contoh Penggunaan

Berwarna,memakai atribut css dan font yang diperbesar

Code : < p style="text-align:center;color: blue;font-size: 24px;">Contoh Penggunaan< /p >
